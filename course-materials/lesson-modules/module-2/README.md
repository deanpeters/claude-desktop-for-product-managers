# Module 2: Understanding the problem space

Turn a messy pile of research into evidence you can defend in any room. No solutions allowed yet. That's the point, and by the end of this module you'll know exactly why.

## Lessons

- `2.1-synthesizing-discovery/`: turning raw research into evidence
- `2.2-market-and-landscape-scan/`: lightweight competitive and market context
- `2.3-segments-and-personas/`: who exactly has this problem
- `2.4-jobs-pains-and-gains/`: what they're trying to get done
- `2.5-problem-framing-canvas/`: clarifying the terrain before solutioning
