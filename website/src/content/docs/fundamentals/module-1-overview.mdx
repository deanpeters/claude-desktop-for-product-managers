The core moves every later module compounds on.

## What you'll cover

- reading and synthesizing files without drowning in output
- asking for structured output you can actually reuse
- making safe, reviewable edits to documents
- verifying results instead of trusting vibes

## The habit this module builds

Ground every claim in a file you can point to, and make Claude separate its evidence from its inference — before you need that discipline on work that matters.
